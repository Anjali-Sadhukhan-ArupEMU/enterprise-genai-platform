sequenceDiagram participant U as User participant FE as React Frontend participant APIM as Azure APIM participant ENTRA as Entra ID participant API as FastAPI Routes participant AUTH as Auth Module participant GOV as Governance Pipeline participant CACHE as Semantic Cache participant CTX as Context Manager participant STORE as Cosmos DB participant ROUTER as Model Router participant PROXY as Azure OpenAI Proxy participant AOAI as Azure OpenAI participant AUDIT as ADLS Audit Logger U->>FE: Type message + select mode FE->>APIM: POST /api/v1/chat?stream=true APIM->>ENTRA: Validate JWT ENTRA-->>APIM: Token valid APIM->>API: Forward with X-User-* headers API->>AUTH: Extract user context (RBAC) AUTH-->>API: UserContext (id, roles, dept) Note over API,GOV: Step 1: Input Governance API->>GOV: check_input(message) GOV->>GOV: PII scan + Injection detection alt Blocked GOV-->>API: BLOCK verdict API-->>FE: Error response else Pass/Warn GOV-->>API: Sanitized content end Note over API,CACHE: Step 2: Cache Lookup API->>CACHE: get(sanitized_text) alt Cache Hit CACHE-->>API: Cached response API-->>FE: Instant response else Cache Miss CACHE-->>API: null end Note over API,STORE: Step 3: Load Conversation API->>STORE: get(conversation_id, user_id) STORE-->>API: Conversation history Note over API,ROUTER: Step 4: Model Routing API->>ROUTER: route(mode, token_estimate) ROUTER-->>API: RoutingDecision (model + fallback) Note over API,CTX: Step 5: Build Context Window API->>CTX: build_messages(conversation, max_tokens) CTX-->>API: Trimmed message array Note over API,AOAI: Step 6: Model Invocation API->>PROXY: stream(model_id, messages) PROXY->>AOAI: Chat Completions (stream=true) AOAI-->>PROXY: SSE token chunks PROXY-->>API: Async iterator API-->>FE: SSE stream (data: chunks) Note over API,AUDIT: Step 7: Post-Processing API->>GOV: check_output(response) API->>STORE: save(conversation + messages) API->>AUDIT: log_audit + log_usage API->>CACHE: set(query, response)
