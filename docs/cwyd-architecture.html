4. Document Ingestion Flow

sequenceDiagram participant ADMIN as Admin User participant UI as Admin Web App participant BLOB as Azure Blob Storage participant QUEUE as Azure Queue Storage participant FUNC as Azure Function App participant DI as Document Intelligence participant EMBED as OpenAI Embeddings participant SEARCH as Azure AI Search ADMIN->>UI: Upload documents (PDF, DOCX, etc.) UI->>BLOB: Store raw files UI->>QUEUE: Add processing message QUEUE->>FUNC: Trigger processing FUNC->>BLOB: Read file FUNC->>DI: Extract text and structure (OCR) DI-->>FUNC: Parsed text + layout Note over FUNC,EMBED: Chunking and Embedding FUNC->>FUNC: Split text into chunks (configurable size + overlap) FUNC->>EMBED: Generate vector for each chunk EMBED-->>FUNC: Chunk embeddings Note over FUNC,SEARCH: Indexing FUNC->>SEARCH: Upload chunks + vectors + metadata SEARCH-->>FUNC: Indexed successfully FUNC-->>UI: Processing complete UI-->>ADMIN: Documents ready for querying
